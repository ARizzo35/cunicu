## wice sync

Synchronize interfaces

### Synopsis

Synchronizes the internal daemon state with the state of the Wireguard interfaces

```
wice sync [flags]
```

### Options

```
  -h, --help            help for sync
  -s, --socket string   Unix control and monitoring socket (default "/var/run/wice.sock")
```
